# Jogo da Forca - PHP (v1.0.0)

## Versionamento Semântico
- **v1.0.0** - Versão estável com todas funcionalidades
- **v0.5.0** - Tratamento de vitória/derrota
- **v0.4.0** - Interface UI básica
- **v0.3.0** - Sistema de tentativas
- **v0.2.0** - Lógica de validação de letras
- **v0.1.0** - Estrutura base do jogo

## Como Executar
1. Inicie o XAMPP/WAMP
2. Coloque a pasta no `htdocs`
3. Acesse `http://localhost/php-version/index.php`

## Tecnologias
- PHP 7.4+
- HTML5/CSS3
- Sessões PHP

## Características
- Lógica server-side
- Estado mantido em sessão
- Page reloads para atualização
